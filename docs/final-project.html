8.0.1 Final Project Solution

The Data:

The cars data sets contain data on specifications of 406 vehicles from 1970 to 1982. Among the variables in the data set are information on fuel consumption (mpg), horsepower, weight, acceleration, origin (Europe, Japan, U.S.), and number of cylinders.

The data set contains categorical variables (such as origin), numerical discrete variables (such as number of cylinders), and continuous variables (such as weight, and acceleration).

Getting Started

  1. Investigate cars_wave1.xls and cars_wave2.xls and prepare the data for SPSS

    1. Remove the first couple rows that contain a heading

    2. Remove the last row that contains summary information

    3. Save and exit

  2. Open SPSS and import cars_wave1.xls and cars_wave2.xls from Microsoft Excel.

    1. Open SPSS

    2. File Open Data Select “Excel” under File Type

    3. Browse for the Excel files and select Open

    4. Keep the box checked for “Read variable names from the first row of data”

    5. Leave the worksheet selected as the default

    6. Select OK

  3. Merge cars_wave1 and cars_wave2 (add cases).

    1. Data Merge Files Add Cases

    2. Select the open data file, then select Continue

    3. The Add Cases dialog will appear

    4. There should not be any “unpaired” variables

    5. Select OK

    6. Your active data file should now have 406 cases

    7. Save this data file and close the “non active” file

  4. Save this new SPSS file!

  5. Using the codebook below, define the proper attributes in Variable View

    1. Be sure to include the missing value code for MPG

    2. You only need to modify the measurement type, variable labels, variable values, and missing values.

Working with Variables:

  1. Recode Origin such that 1=Domestic, 0=Foreign. Remember to recode into a different variable. Give this new variable the proper attributes in variable view.

    1. Transform Recode into different variables

    2. Select Country of Origin (ORIGIN)

    3. Name = NewOrigin

    4. Label = Recode of Origin

    5. Select the Change button

    6. Select the Old and New Values button

    7. Old Value: Value: 1

    8. New Value: Value: 1

    9. Select Add

    10. Old Value: Value: 2

    11. New Value: Value: 0

    12. Select Add

    13. Old Value: Value: 3

    14. New Value: Value: 0

    15. Select Add

    16. Old Value: Value: System or User Missing

    17. New Value: Value: System Missing

    18. Select Add

    19. Select Continue

    20. Select OK

    21. Go to Variable View and enter 1=Domestic, 0=Foreign under Values for this new variable. Also adjust the decimal place to 0.

  2. Convert Miles Per Gallon (MPG) to Liters Per 100 Kilometers

    1. Use the Compute function

    2. The formula to use: LP100K=(100*3.785)/(1.609*MPG)

      1. Transform Compute Variable

      2. Target Variable = LP100K

      3. Numerical Expression: (100*3.785)/(1.609*MPG)

      4. Select OK

      5. Go to Variable View and give this variable a label (Liters Per 100 Kilometers)

  3. Export this SPSS data set to Microsoft Excel (it’s always good to have a back up!). Export all of the variables.

    1. File Save As

    2. Change Files of Type to Excel

    3. Give a name and select location to save

    4. Save

  1. Organize the output by Year (Analyzing groups of cases separately, compare groups). Before proceeding, select only cases with Year not = 0.
